# Ecommerce SQL Database Project

## 📌 Project Overview
This project demonstrates the design and implementation of an E-commerce Database using MySQL. It includes database creation, table relationships, sample data, and SQL queries for business analysis.

## 🛠 Technologies Used
- MySQL
- MySQL Workbench
- SQL

## 📂 Database Tables
- Customers
- Products
- Orders
- Order_Details
- Payments

## 📊 SQL Queries Included
- Customer Order Report
- Revenue by Category
- Product Sales Report
- Payment Status Report
- Top Selling Products
- Monthly Sales Analysis

## 🎯 Skills Demonstrated
- Database Design
- SQL DDL
- SQL DML
- Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- Business Reporting
